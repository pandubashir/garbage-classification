About This Project

Aplikasi ini merupakan portfolio project yang mendemonstrasikan pipeline machine learning secara end-to-end — mulai dari training model hingga deployment ke production. Aplikasi menggunakan Transfer Learning dengan ResNet50 yang telah di-pretrain pada ImageNet, kemudian di-fine-tune pada dataset klasifikasi sampah. Tujuannya adalah menunjukkan bagaimana computer vision dapat mengotomatisasi pemilahan sampah, sebuah permasalahan nyata yang relevan dengan daur ulang dan keberlanjutan lingkungan.
