About This Project

This app is a portfolio project demonstrating end-to-end machine learning — from model training to production deployment. It uses Transfer Learning with ResNet50 pre-trained on ImageNet, fine-tuned on a waste classification dataset. The goal is to show how computer vision can automate waste sorting, a real-world problem relevant to recycling and environmental sustainability.
